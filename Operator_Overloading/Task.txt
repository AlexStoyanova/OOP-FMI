Да се реализира клас за работа с N - мерни математически вектори.
Предефинирайте следните оператори за работа с класа :
- operator+ събира два вектора ( съответно реализирайте += )
- operator- изважда два вектора ( -= )
- operator* - умножава вектор със скалар ( *= )
- operator/ - разделя вектор със скалар ( /= )
- operator% - дава скаларното произведение на два вектора
- operator~ - връща дължината на вектор
- operator|| - определя дали два вектора са успоредни
- operatаtor|= - определя дали два вектора са препендилулярни
